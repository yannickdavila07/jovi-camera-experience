Simulação interativa dos modos de câmera do smartphone Jovi, desenvolvida para as disciplinas de *Front-End* e *Desenvolvimento Web*.

## Funcionalidades

- Login com validação de nome e RM
- Onboarding personalizado (prompt + escolha de perfil)
- Grade de modos de câmera
- Slideshow por modo (informações + câmera real antes/depois)
- Câmera real com captura de foto (antes sem modo, depois com modo)
- Filtros CSS simulando o efeito de cada modo na foto
- Scanner de texto via OCR (Tesseract.js) no Modo Documento

## Tecnologias

- HTML5 semântico
- CSS3 com Flexbox
- JavaScript puro (sem frameworks)
- API navigator.mediaDevices.getUserMedia (câmera)
- Tesseract.js 4 (OCR)
